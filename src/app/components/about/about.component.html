Built for Engineering Excellence.

JobTrackPro isn't just a todo list. It is an enterprise-grade full-stack application I architected to demonstrate modern security, cloud storage, and performance patterns.

Why I built this

Searching for a job is a full-time job. I found that spreadsheets were messy, and generic to-do lists couldn't capture the nuance of interview stages, salary negotiations, and follow-ups.

I built JobTrackPro to bring clarity to the chaos. My goal was to create a centralized hub where I could track every application, visualize my conversion rates, and handle data securely using industry-standard practices.

My Tech Stack

Spring Boot 3

I engineered a robust REST API using Java 21 and Spring Security 6. It features multi-threaded analytics and optimized JPA queries.

Angular 17

I utilized a modern reactive architecture with Angular Signals, standalone components, and TailwindCSS for a fully responsive UI.

Cloudflare R2

I implemented S3-compatible object storage to handle secure file uploads and deliver assets with high performance.

Engineering Challenges I Solved

Hybrid Security

I designed a custom auth system supporting both JWT-based local authentication and OAuth2 (Google/GitHub), allowing seamless account linking.

AI Webhook Pipeline

I architected an inbound email parser using Google Gemini AI. It processes webhooks from Cloudmailin, extracts structured data from unstructured emails, and performs smart upserts to the database.

Atomic Data Integrity

I implemented transactional logic for profile updates. File uploads and text data are handled atomically, and orphaned files in R2 are automatically cleaned up.

Multi-threaded Analytics

I leveraged Java `CompletableFuture` to calculate dashboard analytics in parallel threads, ensuring the dashboard loads instantly even with large datasets.
